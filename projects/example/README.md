# Example Project

This public-safe starter project is safe for first-time users to open in the UI.

- Edit `task.md` to describe the research question you want the agent to work on.
- `memory.md` is optional and ignored by Git. Create it only when you want stable local context.
- Run outputs such as `runs/`, `best_output.md`, `checkpoint.json`, and logs are also ignored.

To create optional local memory for this project:

```bash
cp projects/example/memory.example.md projects/example/memory.md
```
